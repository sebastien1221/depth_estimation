# Depth Estimation Model with addded Uncertainty
Depth estimation with uncertainty trained on NYU Depth V2 Dataset <br>
https://cs.nyu.edu/~fergus/datasets/nyu_depth_v2.html <br>
Indoor Segmentation and Support Inference from RGBD Images <br>
ECCV 2012 <br>

Pretrained RESNET18 base with added bayesian inference for uncertainty estimation <br>
